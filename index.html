Flappy Bird

Created on 11/08/2019 Flappy bird is a project created by Shubhankar Khare.
This game is a clone of a very popular game of 2014 'flappy bird'.
HTML CSS Javascript Bootstrap Jquery has been used to create the game

Here are the rules

he objective is to direct a flying bird
which moves continuously to the right,
between sets of poles. If the bird touches the poles or ground , they lose.
Faby briefly flaps upward each time that the player presses UP key or spacebar
if any of the keys is not pressed, bird falls because of gravity;
